#### Installation.
##### Requirements.
1. cmake
2. make
3. Linux compatible environment.
4. C++ compiler with at least c++17 support.
##### Instructions.
 - Run `git clone https://github.com/gitnold/computer_graphics.git`
 - Navigate to the repo directory and make the build script executable `chmod +x build.sh`
 - Run the build script `./build.sh` from the project root as `./build.sh build` to compile the project and `./build.sh run` to run the compiled binary
